⚪ اربط GitHub أولاً
أي تعديل (إضافة/تعديل/حذف صنف أو قسم، تغيير بيانات الكافيه) بيتنشر على GitHub فوراً عند الضغط على أي زرار حفظ — وهيبان لك في نفس اللحظة "تم النشر ✅" أو رسالة خطأ واضحة "❌" لو حصلت مشكلة.
▾ ⚙ إعدادات الربط مع GitHub غير متصل
⚠️ الـ Token بيتحفظ محلياً في هذا المتصفح فقط (localStorage) ومش بيتبعت لأي حد غيرنا. لما تدوس "حفظ إعدادات الربط" هنختبر الاتصال فوراً ونقولك لو فيه مشكلة في الـ Token أو الصلاحيات.
▾ 📥 استيراد أصناف من ملف CSV
لاستيراد عدد كبير من الأصناف مرة واحدة بدل إضافتهم واحد واحد. أعمدة الملف بالترتيب:
category_ar, category_en, product_ar, product_en, description, price
لو القسم موجود بالفعل (نفس الاسم العربي بالظبط)، الأصناف هتتضاف له. لو مش موجود، هيتعمل قسم جديد.
